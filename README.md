## ⚙️ Tech Stack
- **Language:** Kotlin
- **UI:** XML, ViewBinding
- **Architecture:** MVVM + Clean Architecture
- **Dependency Injection:** Hilt
- **Networking:** Retrofit + Moshi
- **Asynchronous:** Kotlin Coroutines + Flow
- **Image Loading:** Coil
